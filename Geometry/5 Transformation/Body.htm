為了使大家掌握幾何運動的技巧，以下一些題目供大家作練習之用。

1.       設凸四邊ABCD的對角線AC，BD相交於O，且AC⊥BD，已知OA > OC，OB > OD，證BC + AD > AB + CD。

          (第一屆祖沖之數學邀請賽試題)

2.       在△ABC中，D是AB的中點，E、F分別在AC、BC上，證明DDEF的面積不大於△ADE和△BDF面積之和。(第十七屆全蘇數學奧林匹克試題)

3.     在正△ABC中有一點 P 使 PA = 5，PB = 7，PC = 8，求AB。(1985年Iberoamerican MO)

4.     在正方形ABCD內有一點 E，使EA + EB + EC達到最小值，求正方形邊長。

5.     正方形ABCD的邊長為 1，AB，AD上各有一點 P，Q 使 △APQ的周長為 2，求 ∠PCQ。

6.       在 △ABC中，D，E，F分別是 BC，CA，AB 上的點，如果 ，證明AD，BE，CF可形成三角形。

7.       正三角形 △ABC中有一點 P，∠APB = 113°，∠BPC = 123°，證明 AP，BP，CP 可以形成三角形，並求出各內角的度數。

 挑戰題(以下題目的構思皆十分巧妙，難度甚高，但用的都是本文提過的運動，各位可以嘗試挑戰自己的能力)

1.     △ABC為正三角形，D，E，F 分別在 BC，CA，AB上，證明 △DEF的周長 ≧ △ABC周長。

2.       設ABCDEF是凸六邊形，AB = BC = CD，DE = EF = FA，∠BCD = ∠EFA = 60°。G、H是六邊形內兩點使∠AGB = ∠DHE = 120°，

          證明AG + GB + GH + DH + HE ≧ CF。

3.     在一梯形中，AD平行BC，BC = BD = 1，AB = AC，CD < 1，∠BAC + ∠BDC = 180°，求 CD。
